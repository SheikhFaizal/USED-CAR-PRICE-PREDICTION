
# Used Car Price Prediction Analysis

## Situation
The goal of this project is to predict used car prices based on critical features such as vehicle age, fuel type, city, and manufacturer.

## Task
Identify key trends and provide actionable insights for both sellers and buyers using a regression model.

## Action
1. Conducted exploratory data analysis (EDA) to uncover factors influencing car prices.
2. Built a regression model to predict prices with ~70% accuracy (R-squared).
3. Derived actionable insights for stakeholders.

## Result
- Sellers can set competitive prices by leveraging insights on brand and regional demand trends.
- Buyers can identify undervalued vehicles, focusing on features like age and brand depreciation rates.

## Key Features
- Transparent, client-focused model explanation.
- Business-oriented insights tailored for real-world applications.
